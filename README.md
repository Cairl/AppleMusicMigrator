# Apple Music Migrator 🎵

一个自动化的 Apple Music 音乐迁移工具，帮助您快速将歌曲添加到指定歌单。

## ✨ 功能特性

- 🔍 **智能搜索** - 自动搜索歌曲并添加到目标歌单
- 📊 **进度管理** - 实时显示处理进度和预计完成时间
- 🎨 **像素风格 UI** - 复古像素风格界面，支持自定义字体
- 🖼️ **图像识别** - 基于图像识别的自动化操作
- 📝 **状态标记** - 支持多种状态标记（已收录/非原曲/未收录/不收录）
- ⌨️ **鼠标锁定** - 运行时自动锁定鼠标，防止干扰
- 🔄 **自动流转** - 完成一首歌曲后自动处理下一首

## 🚀 使用方法

### 环境要求

- Python 3.8+
- Windows 系统
- Apple Music 或 iTunes 客户端

### 安装依赖

```bash
pip install pandas pyautogui pyperclip pillow pyside6 pywin32 numpy opencv-python
```

### 运行程序

```bash
python music_searcher.py
```

### 配置说明

1. **准备歌单** - 在 `data/歌单.xlsx` 中准备歌曲列表
   - 第一列：歌曲名称（格式：`歌曲名 - 歌手名`）
   - 第二列：状态标记（自动填充）

2. **配置识别库** - 点击「识别库」按钮，截取以下关键图像：
   - 最佳结果标识
   - 歌曲标题锚点
   - 更多选项按钮（···）
   - 添加到播放列表选项
   - 目标歌单名称
   - 跳过重复歌曲按钮（可选）

3. **开始搜索** - 选择搜索模式：
   - **搜索歌名** - 仅搜索歌曲名称
   - **搜索全名** - 搜索完整歌曲名（包含歌手）

### 快捷键

- **ESC / 右键** - 紧急终止当前任务
- **双击预览列表** - 复制歌曲名称

## 🎯 使用技巧

1. **首次使用** - 先运行一次完整流程，确保所有识别图像已保存
2. **批量处理** - 程序会自动流转处理所有未标记的歌曲
3. **手动标记** - 可使用状态标记按钮手动标记歌曲
4. **回退功能** - 使用「上一首」可回退到前一首歌曲

## ⚠️ 注意事项

- 运行时请确保 Apple Music 窗口已打开
- 保持屏幕分辨率不变，避免图像识别失败
- 运行过程中不要手动操作鼠标
- 如遇问题，按 ESC 或右键可紧急终止

## 🛠️ 技术栈

- **Python** - 主要编程语言
- **PySide6** - GUI 框架
- **OpenCV / PyAutoGUI** - 图像识别
- **Pandas** - Excel 数据处理
- **Windows API** - 鼠标钩子和系统操作
